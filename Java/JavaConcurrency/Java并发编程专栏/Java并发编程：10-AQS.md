## AQS是什么

AQS（AbstractQueuedSynchronizer），是JUC中提供的一个抽象类，我们可以通过实现相关抽象方法来快速构造一个同步组件。

ReentrantLock、CountDownLatch、Semaphore 就是AQS的实现。
